Malware Analysis Using VirusTotal (APK)

Overview
This project is part of a learning task focused on understanding malware detection and analysis. A test Android APK file was created for educational purposes and analyzed using VirusTotal to study how malware is identified, classified, and reported by security tools.

Task Objective

The objective of this task is to:

Learn different malware types (virus, worm, trojan, ransomware)

Analyze malware samples using VirusTotal

Understand detection reports and behavior indicators

Study malware lifecycle and spread methods

Identify basic prevention techniques

Summarize findings in a clear and simple way

Tools Used

Kali Linux

Metasploit (msfvenom)

VirusTotal

Methodology

A test APK file was generated for learning purposes.

The APK file was uploaded to VirusTotal.

Detection results from multiple antivirus engines were analyzed.

Malware behavior, classification, and indicators were observed.

Findings were documented and summarized.

Key Findings

VirusTotal detected the APK as Android Trojan malware.

Multiple security vendors flagged the file as malicious.

The malware showed backdoor and remote access behavior.

Even newly created files can be detected by modern security tools.

Disclaimer

This project is strictly for educational and research purposes only. The malware sample was created and analyzed in a controlled environment. No malicious intent or real-world attack is involved.

Conclusion

This task helped in understanding how malware is detected and analyzed using VirusTotal and provided practical insight into Android malware behavior and prevention.
